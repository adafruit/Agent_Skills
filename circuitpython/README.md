# CircuitPython specific skills
For best results update your local AGENTS.md file to include sections about where to find the CIRCUITPY drive, and python virtual environment on your system.

Such as:
````markdown
## Python virtual environment
Your python virtual environment is located at
```
/home/agent/agent_venv
```

## CircuitPython device location
The CircuitPython device appears as a USB drive at
```
/media/username/CIRCUITPY/
```
If you ever see `/path/to/CIRCUITPY/` in skill examples use the location `/media/username/CIRCUITPY/`
````

Replacing `/home/agent/agent_venv` and `/media/username/CIRCUITPY` with the correct filepaths for your system.
